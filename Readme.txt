La idea es que el doctor tenga un arreglo de consumidores o pacientes, estos 
serán guardados solo por el Id para tomarlos de referencia. Cuando este doctor 
le asigne las respectivas comidas, estas se guarden en su respectivo apartado 
en el modelo de "menu". 

El modelo de "menu" tiene un apartado para cada comida del día. Cuando un menu
sea hecho por el doctor este se guardara en el pasiente (Solo se guardara el ID
del menu), para que viendo este que tiene el menu, eliga la comida que quiera 
comer y guardarla en su apartado de menu ya con la comida que selecciono
